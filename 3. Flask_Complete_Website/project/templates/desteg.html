Do you know what is Steganography?

Steganography allows you to encrypt and decrypt message in a image. There are two functions for you to play with on this page. First, when you choose the Steg image, please select one image that is PNG format. The reason it has to be png not jpg because jpg is a compressed image, the program will not work. Once you chose a png image, the next step is enter a message. Then, click upload it will return a image with message encrypted.
When you finish encrypting the image, you can download it and then use the Desteg image function to decrypt it. Amazing right? Find out more about Steganography here: on Wikipedia
